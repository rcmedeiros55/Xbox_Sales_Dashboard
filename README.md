# Xbox Sales Dashboard

## Sobre o Projeto

Este projeto tem como objetivo transformar dados brutos de assinaturas e vendas do ecossistema Xbox em um dashboard visual e organizado, permitindo análises rápidas e suporte à tomada de decisão baseada em dados.

O dashboard foi desenvolvido no Microsoft Excel com foco em:

* Organização visual dos dados
* Monitoramento de indicadores de desempenho (KPIs)
* Análise de vendas e assinaturas
* Visualização clara do comportamento dos clientes

---

## Dados Utilizados

A base de dados contém informações relacionadas às assinaturas Xbox, incluindo:

* ID do assinante
* Nome do cliente
* Plano contratado (Core, Standard e Ultimate)
* Tipo de assinatura
* Renovação automática
* Preço da assinatura
* EA Play Season Pass
* Minecraft Season Pass
* Cupons aplicados
* Valor total da compra

---

## Indicadores (KPIs)

O dashboard apresenta os seguintes indicadores:

### Receita Total

Soma do valor total das assinaturas.

### Total de Assinantes

Quantidade total de registros de assinantes.

### Ticket Médio

Valor médio gasto por assinatura.

### Taxa de Renovação Automática

Percentual de usuários com renovação automática ativada.

---

## Visualizações do Dashboard

### Receita por Plano

Comparativo de faturamento entre os planos Xbox.

### Receita Mensal

Análise temporal do desempenho de vendas.

### Distribuição do Tipo de Assinatura

Visualização da participação entre planos mensais, trimestrais e anuais.

## Como Reproduzir

1. Clone este repositório:

git clone <URL_DO_REPOSITORIO>

2. Abra o arquivo `.xlsx` no Microsoft Excel.

3. Navegue até a aba **Dashboard** para visualizar os indicadores e gráficos.

4. Atualize a base de dados para refletir novos registros.

## Objetivo de Negócio

O principal objetivo é transformar dados operacionais em informações estratégicas, permitindo:

* Monitorar desempenho de vendas
* Identificar planos mais lucrativos
* Analisar comportamento de assinatura
* Apoiar decisões orientadas por dados
